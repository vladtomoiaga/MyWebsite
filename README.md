## Getting Started

### Installation

Below is how you can install and setting up the website.


1. Clone the repo
   ```sh
   git clone https://github.com/vladtomoiaga/MyWebsite.git
   ```
2. Install requirements
   ```sh
   pip install -r requirements.txt
   ```
